Features of the commander

  • Raspberry Pi commander
  • Shows information of multiple Raspberry Pi projects:
  • Boat, Camera and jigging projects
  • 4G connection to internet

Server features

  • Node.js server for web sockets
  • Server can serve multiple boats
  • Simple pub/sub system for passing information between boat and client(s)
  • Serves the web ui html and javascript files
  • Amazon EC2

Web client features

  • Connects to server via web sockets
  • Angular 2
  • Depending of connected project on Raspberry Pi
  • Display motion JPEG feed (if Raspberry Pi has camera)
  • Show and send chat messages
  • Display map if Pi sends GPS coordinates
  • Show controls for boat, camera or jigging device
